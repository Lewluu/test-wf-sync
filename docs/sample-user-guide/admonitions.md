# Notes and admonitions

The following note formats are available with this theme.

## How to format a note

Format a note using the following syntax in markdown:

```markdown
!!! warning
 
    Use three exclaimation points followed by the admonition type. 
    Indent the next line with four spaces to include it in the callout.
```

## Recommended types

!!! note
    This is a note admonition. It is used to provide additional information or context to the reader.

!!! tip
    This is a tip admonition. It is used to provide helpful hints or suggestions to the reader.

!!! warning
    This is a warning admonition. It is used to alert the reader about potential issues or important information.

!!! example
    This is an example admonition. It is used to provide a specific example or case study to illustrate a point.

## Other available types

!!! info
    This is an info admonition. It is used to provide general information or tips to the reader.

!!! important
    This is an important admonition. It is used to highlight critical information that the reader should pay attention to.

!!! success
    This is a success admonition. It is used to highlight successful outcomes or best practices.

!!! failure
    This is a failure admonition. It is used to highlight unsuccessful outcomes or lessons learned.

!!! question
    This is a question admonition. It is used to pose a question to the reader or encourage them to think critically about a topic.

!!! abstract
    This is an abstract admonition. It is used to provide a summary or overview of a topic.

!!! bug
    This is a bug admonition. It is used to report a bug or issue with the product or service.

!!! quote
    This is a quote admonition. It is used to provide a quote or citation from a source.

!!! danger
    This is a danger admonition. It is used to warn the reader about potential risks or hazards.
